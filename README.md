# FastAPI + MongoDB

Basic template to build a high performance/async API with FastAPI and MongoDB.

## Features

- FastAPI
- MongoDB
- JWT and OAuth2 authentication with [FastAPI Users](https://github.com/fastapi-users/fastapi-users)
- Role based authorization
- ToDo's CRUD
- Pagination
- Docker support
