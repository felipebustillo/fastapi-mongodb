# FastAPI + MongoDB

Basic template to build a high performance/async API with FastAPI and MongoDB.

## Features

- FastAPI
- MongoDB
- JWT and OAuth2 authentication
- Beanie ODM
- Email verification
- Role based authorization
- Pagination
- Docker support
